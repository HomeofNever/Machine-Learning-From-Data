Lecture Information¶

People¶

  • Instructor: Uzma Mushtaque

  • TAs and programming mentors: See course Syllabus

Learning Outcomes¶

  1. Develop an understanding of the limits to which information can be learned from data, and how.

  2. Be able to: implement, as well as critique basic models of learning from data.

  3. Formulate a learning problem precisely, in terms of inputs and outputs; select a learning model and algorithm, run it on the data, and interpret the results; provide some measures of how effective the learning was

  4. Demonstrate an understanding of application of ML to real-world problems.

Textbook¶

  • Learning From Data by Abu-Mostafa, Magdon-Ismail, Lin.

    • Available in e-book form
